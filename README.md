# Practice javascript

## Stack

- Node v12.16.2
- Javascript

### Clone repo

```bash
# Clone repo
$ git clone https://github.com/WilliamPerezBeltran/practiceJavascript.git

# Go to directory
$ cd practiceJavascript

# Install depedencies
$ yarn install or npm install
```

### Run test

```bash
# Run test
$ npm run test
```
